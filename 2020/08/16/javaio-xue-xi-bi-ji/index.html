javaio学习笔记一

javaio
发布日期:   2020-08-16
文章字数:   2.1k

io文件对象及一些常用方法

创建文件对象,这里的文件好像不需要实际存在

File f=new File("d:/lol.txt");

直接输出f是文件对象的名称

System.out.println(f);

查看文件是否存在

System.out.println(f.exists());

判断是不是文件夹

System.out.println(f.isDirectory());

判断是不是文件,这地方判断不是文件夹就是文件

System.out.println(f.isFile());

查看文件长度

System.out.println(f.length());

查看文件最后修改时间，这个方法返回一个long型的数据，不用日期方法格式化的话就会输出0，像下面这样，如果直接输出time的话就是0,我原以为是时间戳，现在看来好像不是

long time = f.lastModified();
Date d = new Date(time);
System.out.println(d);

设置最后修改时间0，指的是1970/1/1 8:00:00

f.setLastModified(0);

重命名文件，先创建新的文件对象，然后用renameTo方法重命名，如果文件类型不同没法重命名，比如想把txt文件重命名为exe文件是不可以的

File f2 =new File("d:/yeah.txt");
f.renameTo(f2);

list()方法返回当前文件夹下的所有文件和文件夹，list方法返回的是一个String数组；这里如果是一个文件夹的话会返回数据，如果是文件的话会返回null；

String a[]=new String[100];
a=f.list();
String hhh= Arrays.toString(a);
System.out.println(hhh);

listFile()方法返回的是一个文件数组，当前文件对象下的所有文件，如果f不是文件夹的话会抛出空指针异常；

File f = new File("d:/");
File[] fs= f.listFiles();
for(int i=0;i<fs.length;i++)&#123;
    System.out.println(fs[i]);
&#125;

getParent()方法返回一个字符串，表示文件对象f所在的文件夹

String a = f.getParent();
System.out.println(a);

getParentFile()方法返回一个文件对象，代表f所在的文件夹

File a = f.getParentFile();
System.out.println(a);

mkdir()方法，在当前文件对象所在的文件夹下创建当前文件对象，有点绕，文件对象不一定存在的，比如说d盘下边没有softwares这个文件夹，我创建一个文件对象然后调用mkdir()方法就可以新建一个文件夹，当然前提是他的父目录也就是d盘要存在。

File f = new File("d:/softwares");
f.mkdir();

createNewFile()方法，创建一个空文件，像下面这个就会在softwares下面创建一个叫sss的空文件，当然，你文件对象里边有后缀的话就会创建相应的类型文件，这里如果父目录不存在就会产生异常，所以通常会同时产生父目录。

File f = new File("d:/softwares/sss");
f.createNewFile();
f.getParentFile().mkdirs();//这条语句会把文件对象路径上所有的父目录都创建出来

listRoots()方法会列出系统中所有可用的根盘符，就是c:/这种，返回的是一个文件数组

File[] a = f.listRoots();

delete()方法，删除文件，如果不存在这个文件的话也不会抛出异常，存在的话就会删除

f.delete();

虚拟机结束之后删除文件，通常可以用来删除临时文件

f.deleteOnExit();

IO字节流

InputStream是抽象类，不提供具体实现，FileInputStream是他的子类

文件输入流获取文件对象，通过read()方法将流写入a[]，然后输出的是ascⅡ码值，最后记得手动关闭一下输入流

File f = new File("d:/sss.txt");
FileInputStream fis = new FileInputStream(f);
byte[] a=new byte[(int)f.length()];
fis.read(a);
for(int i=0;i<a.length;i++)&#123;
    System.out.println(a[i]);
&#125;
fis.close();

文件输出流获取文件对象，通过write()方法将a[]写入流，这里的文件对象如果不存在会自动创建，如果父目录不存在的话会抛出异常；

File f = new File("d:/lalala.txt");
FileOutputStream fos = new FileOutputStream(f);
byte a[] = &#123;67, 73&#125;;
fos.write(a);
fos.close();

关闭流的方式

标准的严谨的关闭流的方式，在try外边声明流，如果没有关闭流，在finally中始终会去判断流是否关闭了，如果是在try内声明流并且关闭的话，如果在关闭之前catch到了异常，这个流就不会关闭了，容易造成资源占用；

File f = new File("d:/lalala.txt");
FileOutputStream fos = null;
try&#123;
    fos=new FileOutputStream();
&#125;catch (IOException e)&#123;
    e.printStackTrace();
&#125;finally&#123;
    if(null!=fos)&#123;
        try&#123;
            fos.close();
        &#125;catch (IOException e)&#123;
            e.printStackTrace();
        &#125;
    &#125;
&#125;

将流声明在try()中，在try，catch或者finally结束时会自动关闭流；

File f = new File("d:/lalala.txt");
try(FileOutputStream fos = new FileOutputStream(f))&#123;
&#125;catch (IOException e)&#123;
    e.printStackTrace();

使用FileReader类进行文件读取，这个是字符流，所以用字符char数组来接收；

try (FileReader fr = new FileReader(f)) &#123;
    char a[] = new char[(int) f.length()];
    fr.read(a);

IO缓存流

字符流和字节流都存在弊端，io操作过于频繁，对外存造成访问压力，使用缓存流可以解决这个问题；

缓存字符流类BufferedReader，必须基于一个已经存在的流进行实例化，readLine()方法用于读取一行数据；

try (FileReader fr = new FileReader(f);BufferedReader br=new BufferedReader(fr)) &#123;
    while(true)&#123;
        String line=br.readLine();
        if(null!=line)&#123;
            System.out.println(line);
        &#125;
    &#125;
&#125; catch (IOException e) &#123;
    e.printStackTrace();
&#125;

缓存字符输出流PrintWriter，一次可以输出一行，同样必须已经存在流；

try (FileWriter fw = new FileWriter(f);PrintWriter pw=new PrintWriter(fw)) &#123;
    pw.println("jjgdjsgdhfsfjhjgajhfafa");
    pw.println("dhhsdjgdgadgasd");
    pw.println("jsjdgsgdsadgasdgdgas");
&#125; catch (IOException e) &#123;
    e.printStackTrace();
&#125;

flush()方法，强制清空缓存区，将缓存中的内容全部写入外存；

File f = new File("d:/sss.txt");
try (FileWriter fw = new FileWriter(f);PrintWriter pw=new PrintWriter(fw)) &#123;
    pw.println("jjgdjsgdhfsfjhjgajhfafa");
    pw.flush();
    pw.println("dhhsdjgdgadgasd");
    pw.flush();
    pw.println("jsjdgsgdsadgasdgdgas");
